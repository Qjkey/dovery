# Приветствую в репозитории Dovery!

**Dovery** — это мессенджер со сквозным шифрованием. В этом репозитории находится весь его исходный код.

> Мой текущий опыт не всегда позволяет писать безупречный код. Если вы обнаружите ошибки, уязвимости или недочеты, пожалуйста, сообщите мне об этом в Telegram [@drok_off](https://t.me/drok_off). Буду очень благодарен!

# Стек технологий

* **Фронтенд** Чистая верстка (HTML/CSS/JS) с использованием собственного UI-кита [TMA ui kit 3.0](https://github.com/Qjkey/tma-ui-kit-3-0)
* **Бэкенд** Python, фреймворк **Flask** и **Flask-SocketIO** для работы в реальном времени
* **База данных** SQLite3 (в будущем, при масштабировании проекта, планирую изучить и переехать на PostgreSQL)
